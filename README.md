Scholar –Mentorship Platform
Description:
Scholar is a cutting-edge web-based mentorship platform designed to bridge the gap between
aspiring learners and experienced professionals. It provides a seamless and structured
environment for discovering mentors, scheduling sessions, real-time communication, and
tracking learning progress. Built with a modern frontend and a scalable backend, Scholar
delivers a professional-grade mentorship experience through intuitive design, secure
authentication, and real-time data handling.

Key Features:
• User Profiles for mentors and mentees
• Session scheduling and management
• Real-time chat and messaging
• Progress and goal tracking
• Secure authentication

Tech Stack:
Vite + React + TypeScript, Tailwind CSS, shadcn/ui, Supabase (Auth + Realtime DB +
Storage), Node.js & Express (planned), PostgreSQL
